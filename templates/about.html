About This Project

EduPredict AI is an end-to-end Machine Learning web application developed using Flask. It predicts a student's Mathematics Score based on demographic and academic information.

Project Overview

This application uses a trained Machine Learning model to estimate a student's Mathematics score based on factors like gender, parental education, lunch type, test preparation, reading score, and writing score.

Technologies Used

Python

Core programming language used for backend development.

Flask

Web framework used to build the application.

Machine Learning

Linear Regression model trained on student performance data.

Pandas & NumPy

Used for preprocessing and handling the dataset.

Dataset Features

  • Gender
  • Race / Ethnicity
  • Parental Level of Education
  • Lunch Type
  • Test Preparation Course
  • Reading Score
  • Writing Score
